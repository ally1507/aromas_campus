## Aromas campus

**Versión 2026 — python**

Una API desarrollada en python para gestionar el control de inventarios en la cafeteria, registrando entradas, salidas, y ajustes de disponibilidad. Diseñada para entornos empresariales que requieren trazabilidad completa de movimientos de stock.

---

## 🧩 Problemática que resuelve

Cafeteria con la necesidad:
- Registrar movimientos de productos (entradas, salidas, ajustes).
- Validar operaciones según disponibilidad y estado.
- Evitar inconsistencias en stock ante errores o anulaciones.

---

## ✨ Funcionalidades principales

- Gestión de productos.
- Registro de disponibilidad
- Agregar y quitar productos
